To run locally:

- clone Repo
- from the root folder, run `python3 -m venv .venv` from the command line
- run `source .venv/bin/activate` from the command line
- run `pip install -r requirements.txt` from the command line
- run `flask --app app run` from the command line
- The terminal should tell you that the app is running on http://127.0.0.1.5000 - Copy this URL
- Open a new terminal at the root directory
- run `npm install`
- run `npm run build`
- Open the copied URL in your browser
